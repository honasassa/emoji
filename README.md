emoji
======================

commandline tool for emoji

install
======================

```
go install github.com/kkkkkt/emoji-commandline-tool
```

and be sure your PATH includes$GOPATH/bin:

```
export PATH=$PATH:$GOPATH/bin
```

use
======================

* emoji list(l)
  * show emoji list

* emoji search(s) query
  * search emoji

* emoji random(r)
  * show emoji in random
